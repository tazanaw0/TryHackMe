
| Command | Purpose                                                             |
| ------- | ------------------------------------------------------------------- |
| touch   | Create file                                                         |
| mkdir   | create folder                                                       |
| cp      | copy file or folder                                                 |
| mv      | move file or folder + renaming                                      |
| rm      | remove file or folder (must include -R switch before file/dir name) |
| file    | determine type of file                                              |
